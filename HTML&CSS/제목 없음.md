## HTML/CSS



* HTML : markup Language
  * Markup Language : 자료의 구조를 표현하기 위한 언어
    									자료를 어디에 어떻게 배치할지 결정
* `<p></p>` : p태그 (paragraph, 본문)
* `<h1></h1>`: h태그 (heading, h1 ~ h6)
* `<ul>, <ol>, <li>`: unordered, ordered(자동번호), list item 태그
* 1. 모든 요소는 tag안에 넣을 것
  2. 일부 tag는 속성 가짐
  3. 태그 안에 태그 가능
